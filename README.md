# Project Overview
This project analyzes sales performance using SQL queries and Power BI dashboards.
The goal is to understand revenue trends, customer behavior, product performance, and operational metrics such as return rates and transaction volumes.
The analysis supports data-driven decision making by highlighting key insights across weekly revenue, VIP contribution, discount effectiveness, and overall revenue.
